# DVLD - Driving & Vehicle License Department System

A desktop application developed using C# and Windows Forms for managing driving licenses, applications, drivers, tests, and users.

## Features

* User Management
* Driver Management
* License Management
* International License Management
* Local Driving License Applications
* Detained Licenses Management
* License Renewal
* License Replacement
* Test Appointments Management
* Permissions & Role-Based Access Control

## Technologies Used

* C#
* .NET Framework
* Windows Forms
* SQL Server
* ADO.NET
* Stored Procedures
* Three-Tier Architecture
* Object-Oriented Programming (OOP)
* SHA-256 Password Hashing

## Project Structure

* DVLD Project Final (Presentation Layer)
* DVLD_Business (Business Layer)
* DVLD_DataAccess (Data Access Layer)
* clsSecurity (Security Library)

## Database
